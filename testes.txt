// Vitória de X (Horizontal)
XXXOOVVVV // X na primeira linha
OVOXXXOVV // X na segunda linha
OOVVVXXXX // X na terceira linha

// Vitória de X (Vertical)
XOVXOVXOO // X na primeira coluna
OXVVXVOXV // X na segunda coluna
OVXVVXOVX // X na terceira coluna

// Vitória de X (Diagonal)
XVOVXOVVX // X na diagonal principal (topo-esq para baixo-dir)
OVXOXVOXV // X na diagonal secundária (topo-dir para baixo-esq)

// Vitória de O (Horizontal)
OOOXXVXVV // O na primeira linha
XXVOOOXVV // O na segunda linha
XVXVXOOOV // O na terceira linha

// Vitória de O (Vertical)
OXXOXVXOV // O na primeira coluna
XOXOXVOXV // O na segunda coluna
XXOOXOVXO // O na terceira coluna

// Vitória de O (Diagonal)
OXXVXOXVO // O na diagonal principal (topo-esq para baixo-dir)
XXOOXOXOV // O na diagonal secundária (topo-dir para baixo-esq)

// Exemplos adicionais (Empate e Jogo em Andamento)
XOXOXOXXO // Empate (tabuleiro cheio, sem vencedor)
XOVVOXVVX // Empate (tabuleiro cheio, sem vencedor)
XOVOXVXVV // Jogo em andamento (ninguém ganhou, tabuleiro não está cheio)
VXVXVXVXV // Jogo em andamento (tabuleiro quase vazio)

XOZOXOXOX   // Inválido: caractere inválido (Z)
